# LangChain 系统性自学课程大纲

## 第一阶段：基础入门（1-2 周）

### 模块 1.1：环境搭建与快速入门
- LangChain 生态系统概述
- 安装配置（`pip install langchain`）
- 第一个 Hello World 应用
- 基础 LLM 调用与消息格式
- 流式输出基础

### 模块 1.2：核心组件 - 模型与消息
- Chat Models 与 LLM Models 的区别
- Message 类型：`HumanMessage`、`AIMessage`、`SystemMessage`
- 模型选择与配置
- 结构化输出（Structured Output）
- 工具调用基础（Tool Calling）

### 模块 1.3：核心组件 - 工具（Tools）
- 工具定义与装饰器
- 内置工具与自定义工具
- 工具错误处理
- 工具流式输出

### 模块 1.4：核心组件 - 提示词工程
- 静态提示词模板
- 动态提示词
- 前后模型钩子（Pre/Post-model hooks）
- 中间件系统（Middleware）

- **RAG Agent 实现**
  - 检索增强生成流程
  - 自定义 RAG Agent（LangGraph 实现）
  - 检索优化策略

### 模块 2.3：数据库交互 - SQL Agent
- SQL Agent 架构
- 数据库连接与安全
- 人类介入审核（Human-in-the-loop）
- **自定义 SQL Agent**（LangGraph 原生实现）

### 模块 2.4：语音 Agent
- 语音输入输出处理
- 实时交互设计
- 音频流处理

### 模块 2.5：上下文工程（Context Engineering）
- 静态运行时上下文
- 动态运行时上下文（State）
- 跨对话上下文（Store）
- 可变性与生命周期管理

## 第四阶段：LangGraph 深度掌握（2-3 周）

  - 状态图（StateGraph）
  - 消息图（MessageGraph）
- 函数式 API
  - 单函数代理构建
  - 入口函数设计
- 思考方式：Thinking in LangGraph

### 模块 4.2：图的高级用法
- 子图（Subgraphs）
  - 图嵌套与隔离
  - 子图状态传递
- 时间旅行（Time Travel）
  - 状态快照
  - 状态回滚与重放

### 模块 4.3：持久化与流式
- 检查点系统（Checkpointer）
  - 状态持久化
  - 内存添加与管理
- 流式输出
  - 事件流
  - 实时更新
- 图执行前端集成

### 模块 4.4：LangGraph 实战
- **自定义 RAG Agent**
  - 使用 LangGraph 原语构建
  - 细粒度控制检索流程
- **自定义 SQL Agent**
  - 最大灵活性的数据库交互
